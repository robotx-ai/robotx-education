# OriginMan User Guide

The following operations assume some basic robot development knowledge. It helps to know Python, Linux, Ubuntu, SSH, and ROS or ROS2 before moving deeper into the system.

![OriginMan core platform](/assets/courses/originman/originman-core.jpg)

## Power On and Network Connection

After opening the OriginMan package, you will see the robot itself. Every OriginMan ships with its own ready-to-run factory image.

Turn on the power switch at the lower back of the robot to start the system.

When startup is successful, the chest light ring changes from white to colorful, then OriginMan says, "Hello, I'm OriginMan." That means the image has started correctly.

If the robot is connected to a network, the OLED screen on the back shows the IP address you need for connection. If it is not connected yet, use an ethernet cable to configure networking between OriginMan and your local computer first.

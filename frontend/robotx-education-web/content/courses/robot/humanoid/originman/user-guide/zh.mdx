# OriginMan 用户指南

以下操作需要一定的机器人开发基础。建议在继续之前先了解 Python、Linux、Ubuntu、SSH 以及 ROS 或 ROS2 的基本知识。

![OriginMan 核心平台](/assets/courses/originman/originman-core.jpg)

## 开机与网络连接

打开 OriginMan 包装后，你会看到机器人本体。每一台 OriginMan 都预装了可以直接运行的出厂镜像。

通过切换机器人背部下方的电源开关即可启动系统。

成功启动后，胸口灯环会由白色变为彩色，随后 OriginMan 会播报 “Hello, I'm OriginMan”。这表示镜像已经正确启动。

如果机器人已连接网络，背部 OLED 屏会显示用于连接的 IP 地址。如果尚未联网，请先使用网线在 OriginMan 与本地电脑之间完成网络配置。

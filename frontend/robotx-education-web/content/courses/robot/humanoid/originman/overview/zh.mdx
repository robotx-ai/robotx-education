# OriginMan

OriginMan 是一款面向未来的智能桌面人形机器人，同时也是首个全开源的 ROS2 桌面人形机器人开源项目，旨在让每一位参与者都能享受机器人开发的乐趣。

![OriginMan 全身平台](/assets/courses/originman/originman-1.png)

## 开源共建

“开放共享，智创未来”是 OriginMan 的进化哲学。它不是封闭产品，而是期待与全球开发者一起共建机器人未来的开放平台。

### 开源生态

OriginMan 提供主控系统源码、大脑小脑通信协议以及动作编辑软件源码，帮助每位开发者轻松打造专属的“智能大脑”，释放独特创意，开发多样化应用。

### 持续迭代

无论真理多么深远，每一次小小的进步都值得欣喜。依托强大的社区支持，OriginMan 与开发者共同前行，软硬件持续迭代，版本周期更新，一起成长壮大。

### 畅享 AI 开发

OriginMan 出厂即支持诸多 AI 功能，并提供 DeepSeek、Qwen 等模型 API 及使用案例，让你的机器人开发之旅拥有更多可能性。

### 内容齐全

OriginMan 不仅是一个开源套件，更像你的机器人伙伴。硬件、软件、图文教程、视频资源、论坛支持一应俱全，丰富的软硬件支持帮助你更快成长。

## 智能进化平台

OriginMan 不仅是一个硬件开发平台，更是一个不断进化的智能算法平台，你可以在它之上持续拓展更多能力。

### 认知核心

OriginMan 原生搭载地瓜机器人开发板 RDK X5。10 TOPS 算力支持实时视觉推理与自然语言处理。预装的大模型交互系统可实现多轮对话理解、AI 智能互动等能力，让“智能机器人”开发真正落地。

### 应用丰富

OriginMan 采用 ROS2 及 TogetheROS.Bot（地瓜机器人深入优化的 ROS 开源机器人操作系统）作为机器人软件系统，兼容 ROS 生态中的众多应用，即装即用。原装应用覆盖人体模仿学习、视觉巡线、AI 语音交互等方向，方便快速上手开发。

## 始于初心

每个机械关节都承载着对智能体的想象，每次代码提交都在描绘人机共生的未来。

### 携手同行

这里没有孤胆英雄，只有携手同行的探索者。无论你是第一次接触机器人，还是已经拥有丰富开发经验，都能在社区中找到志同道合的伙伴。

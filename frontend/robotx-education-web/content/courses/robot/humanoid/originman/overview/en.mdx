# OriginMan

OriginMan is a future-facing intelligent desktop humanoid robot and the first fully open-source ROS2 desktop humanoid project. It is built so every participant can enjoy the process of robot development instead of treating robotics as a closed system.

![OriginMan full body platform](/assets/courses/originman/originman-1.png)

## Open Source, Built Together

“Open sharing, intelligent creation, future together” defines the OriginMan philosophy. The project is designed to evolve with developers around the world rather than behind a closed product wall.

### Open Ecosystem

OriginMan provides source code for the main control system, communication protocols between the upper and lower control layers, and motion editing software. That gives each developer the foundation to build a custom “intelligent brain,” test original ideas, and create diverse robot applications.

### Continuous Iteration

Even small progress matters when the goal is long-term robotics innovation. With community participation, both the hardware and software continue to iterate, and each version cycle expands what developers can build.

### AI Development Out of the Box

OriginMan ships with built-in AI capability and practical examples for models such as DeepSeek and Qwen. That lowers the barrier to experimentation and gives developers more room to explore real interactive robot behavior.

### Complete Learning Support

OriginMan is more than an open-source kit. It is a complete development companion with hardware, software, illustrated guides, video resources, and community discussion support so builders can move faster with less friction.

## An Intelligent Evolution Platform

OriginMan is not only a hardware platform. It is an evolving intelligent algorithm platform designed for extension and experimentation.

### Cognitive Core

OriginMan uses the D-Robotics RDK X5 robot development board as its native compute platform. With 10 TOPS of onboard performance, it supports real-time vision inference and natural language processing. The preinstalled large-model interaction system makes multi-turn dialogue, AI interaction, and “intelligent robot” development directly accessible.

### Rich Application Layer

OriginMan runs on ROS2 and TogetheROS.Bot, D-Robotics’ deeply optimized open-source robotics operating system. It stays compatible with many ROS ecosystem applications and supports immediate experimentation in areas like human imitation learning, vision-based line following, and AI voice interaction.

## Built From First Principles

Every mechanical joint carries an idea about embodied intelligence. Every code commit adds another line to the future of human-robot collaboration.

### Walk Together

This is not a project for isolated heroes. It is a platform for people building together. Whether you are just entering robotics or you already have development experience, the community is intended to give you peers, references, and momentum.

### OriginMan Main Controller (`RDK X5-8GB`)

D-Robotics `RDK X5` uses the `Sunrise 5` intelligent computing chip and delivers up to `10 TOPS` of compute performance. It is a general-purpose development kit designed for intelligent computing and robotics applications. The board offers rich interfaces, a development-friendly setup, and support for complex models and modern algorithms such as `Transformer`, `RWKV`, `Occupancy`, and `Stereo Perception`, helping intelligent applications move into practice faster.

- CPU: `Cortex-A55` 64-bit `1.5GHz`, 8 cores
- GPU: `32 GFLOPS`
- BPU: `1XBayes-e@10TOPS`
- Memory: `8GB`
- System storage: `Micro SD` card slot
- USB ports: `1` USB `2.0` (`Type-C`), `4` USB `3.0` (`Type-A`), and `1` debug serial port (`Micro USB`)
- Bluetooth: `Bluetooth 5.4`
- Wi-Fi: `802.11n`, `2.4GHz/5GHz` dual-band `Wi-Fi 6`

![OriginMan RDK X5-8GB](/assets/courses/originman/basic/rdk-1.png)

### Cerebellum Baseboard

- Main control chip: `MM32F103CBT6`, with a CPU speed of up to `96 MHz`
- IMU chip: built-in `6-axis IMU` for real-time posture and acceleration sensing
- Interfaces: bus servo, `IIC`, `GPIO`, and other interfaces, with support for `IIC` protocol communication
- Additional onboard resources: built-in AI voice processing chip and `MEMS` digital-output microphone
- Built-in audio amplifier: output power `4Ω 3.2W @5V`
- System memory: `32GB`
